MT5 BACKTEST AUTOMATION PACKAGE V1.1.5

REQUIREMENTS:
- MetaTrader 5 installed
- Expert Advisors (.ex5 files) in the specified folder
- Sufficient disk space for reports

CONFIGURATION:
Before running, update these variables in the flow:
- mt5Path: Path to your MetaTrader 5 executable
- eaPath: Path to your Expert Advisors folder
- reportPath: Where you want reports saved

USAGE:
1. Import this flow into Power Automate Desktop
2. Modify the configuration variables as needed
3. Run the flow
4. Reports will be saved in the specified folder
